# ATM Management System using Python

## Overview
This is a simple ATM Management System developed using Python. It simulates basic ATM operations through a menu-driven program.

## Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- Exit System
- User-Friendly Menu

## Concepts Used
- Functions
- While Loop
- Conditional Statements (if-elif-else)
- User Input
- Global Variables
- F-Strings

## Technologies Used
- Python 3
- VS Code

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Run the Python file

```bash
python ATM_MANAGEMENT_SYSTEM_PROJECT.py
